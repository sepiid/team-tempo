Bulleted Lists

CSS selector:

ul,li

Sample code:

<ul><li>Unordered List Item1 </li><li>Unordered List Item2 </li></ul>

Rendered element:

  • Unordered List Item1
  • Unordered List Item2

Numbered Lists

ol,li

Sample code:

<ol><li>Ordered List Item1 </li><li>Ordered List Item2 </li></ol>

  1. Ordered List Item1
  2. Ordered List Item2
